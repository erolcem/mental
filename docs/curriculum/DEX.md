# Curriculum Review — DEX (Dexterity)

**Status: PROPOSAL for Erol's markup.** Accessibility notes specific to this
stat: ASE *certification* legally requires 2 years' work experience — but the
practice exams are public, so the tree tests knowledge with them and proves
hands with real teardowns anyone can buy. Memory sport has a genuine online
arena (Memory League) — exam mimicry solved. Karate is the one skill that
inherently needs partners; the tree keeps dojo gradings (open to anyone, low
cost) but every kata/kihon node is video-verifiable against JKA references.
